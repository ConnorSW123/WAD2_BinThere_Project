Can I add a new bin type?

What is Upvote and Downvote?

It is a interactive system to help us see if there is a problem with the bin location or not. Upvote if everything was okay and Downvote if there are problems. Join us to improve the map!

What is the website section in the profile?/ Do I need to add a website?

No, you don't have to. However if you do want to, you can add your social media link or even a link to your favourite shop that commits to sustainability!
